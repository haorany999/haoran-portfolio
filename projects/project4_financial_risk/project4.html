Project Overview

This project focuses on developing a robust ETL (Extract, Transform, Load) pipeline for financial risk data, performing comprehensive risk analysis, and creating a web application to visualize the results. The project utilizes various data sources, including the S&P 500 companies, and implements machine learning models for predictive analysis.

Key Skills Utilized:

  • Data Analytics: Python, RStudio, Tableau, Microsoft Office Suite (Expert in Excel), Bloomberg Terminal.
  • Database Mgmt: MySQL, PostgreSQL (pgAdmin), NoSQL (MongoDB, PyMongo, Neo4j).
  • Machine Learning: Python (Scikit-learn, Pipeline, TensorFlow), R (Feature Selection, Clustering, Regression).

Methodology

  1. Data Extraction from Multiple Sources (Yahoo Finance API)
  2. Data Storage in MongoDB (NoSQL database)
  3. Data Transformation and Loading into PostgreSQL (Relational database)
  4. Data Analysis and Visualization
  5. Machine Learning Model Implementation (Decision Tree and Linear Regression)
  6. Web Application Development with Flask

Step 4: Machine Learning Model Implementation


def train_test_model(df):
    X, y = prepare_data(df)
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)
    clf = DecisionTreeClassifier(max_depth=3, random_state=42)
    clf.fit(X_train, y_train)
    y_pred = clf.predict(X_test)
    return accuracy_score(y_test, y_pred)

# Example usage
df = get_data('AAPL')
accuracy = train_test_model(df)
print("Accuracy of decision tree model for AAPL: {:.2f}%".format(accuracy * 100))

This step implements a decision tree classification model to predict stock price movements.
