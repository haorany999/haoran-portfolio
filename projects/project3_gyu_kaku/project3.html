Overview

This project aims to optimize revenue for Gyu-Kaku restaurants by analyzing the impact of different operational strategies. We focus on three key areas: ordering methods, lighting types, and takeout options.

Data Description

We simulated data for 10 restaurants over 22 days, considering various factors such as:

  • Ordering methods: À la carte vs. Buffet
  • Lighting types: Warm, Cold, and Natural
  • Takeout options: With and Without Takeout

Methodology

We used R for data simulation and analysis. Here's a snippet of our code for data generation:


library(data.table)

# Parameters Initialization
restaurants      <- 10         # Number of restaurants
days             <- 22         # Number of days for the simulation
baseline_revenue <- 5625       # Baseline daily revenue
std_dev          <- 500        # Standard deviation for revenue
effect_size      <- 562.50     # 10% increase in revenue

# Create a data table for ordering methods
ordering_data <- data.table(
  Day = rep(1:days, times = restaurants),
  Restaurant_ID = rep(1:restaurants, each = days)
)

# Generate revenue data for each ordering method
ordering_data[, `:=`(
  Revenue_A_la_carte = round(rnorm(n = .N, mean = baseline_revenue, sd = std_dev), 1),
  Revenue_Buffet      = round(rnorm(n = .N, mean = baseline_revenue + effect_size, sd = std_dev), 1)
)]

# Display the ordering data in an interactive table
datatable(data = ordering_data)

This code snippet demonstrates how we generated simulated data for different operational scenarios, including ordering methods, lighting types, and takeout options.

Results

Our analysis revealed:

  • À la carte ordering method showed a potential increase in revenue compared to the Buffet method, with an average effect size of 198.33 units.
  • Implementing takeaway services demonstrated a positive impact on revenue, with an average effect size of 673.4 units.
  • Both findings showed high statistical power, indicating reliable results.

This chart visualizes the revenue comparison between different operational strategies.

Conclusion

Based on our findings, we recommend:

  • Implementing À la carte ordering as the primary method to potentially increase revenue.
  • Expanding takeaway services to tap into additional revenue streams.
  • Conducting further research to validate these findings in real-world settings across various locations.
  • Considering additional factors such as customer satisfaction, food quality, and operational costs in future analyses.
